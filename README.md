
# 🧮 Calculator Project

A fully functional calculator built using **HTML, CSS, and JavaScript**.  
This project demonstrates handling user interactions, DOM manipulation, and implementing real-time logic in a clean, responsive interface.

---

## ✨ Features
- Clean and responsive UI
- Interactive button handling
- Real-time calculations using JavaScript
- Clear (`C`) button to reset input
- Error handling for invalid expressions
